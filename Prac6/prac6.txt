JSONObject comes from the json-simple or org.json library, which is not included by default.

Option 1: Download and Add json-simple.jar (Recommended)
Download the json-simple library:

Go to Maven Repository: JSON-Simple
Download json-simple-1.1.1.jar.
Add to Eclipse Project:

Right-click on your project in Eclipse → Build Path → Configure Build Path.
Go to the Libraries tab → Click Add External JARs....
Select json-simple-1.1.1.jar and click Apply and Close
